Configuration requise 

  Java 8 avec Maven 3+

Instalation

  $ mvn clean install

Lancement

  $ java -cp discount-basket-1.0.jar com.company.basket.Main

Astuce

  Pour itérer autant de fois que nécessaire : $ cd .. && mvn clean install && cd - && java -cp discount-basket-1.0.jar com.company.basket.Main
